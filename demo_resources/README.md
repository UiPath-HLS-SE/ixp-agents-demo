# Demo Resources

These files are synthetic examples meant for demos and local experimentation.

Files:

- `synthetic_packet.txt`: a text-only stand-in for packet content
- `synthetic_ixp_output.json`: example baseline extracted output
- `synthetic_ground_truth.jsonl`: example expected values for evaluation
- `synthetic-ixp-walkthrough/`: shareable scorecards, workbooks, manifests, and a static dashboard for a baseline IXP vs. second-pass walkthrough
- `kswic-payer-correspondence-demo/`: synthetic payer correspondence PDFs, IXP-style extraction payloads, Maestro routing artifacts, and dummy automation contracts for a Wichita intake demo

They are intentionally generic and safe to share.
