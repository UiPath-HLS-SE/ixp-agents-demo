Sanitized And Shareable

Coding agents for IXP automations, packaged as a clean demo repo.

This repo is built to show how coding agents help developers build better document solutions faster: faster artifact creation, faster extraction troubleshooting, and a much more realistic path for Studio-first teams to create useful Python logic such as coded agents and automation helpers.

Open Full Presentation View On GitHub
Key Message

Coding agents accelerate IXP work in two major ways: they help developers build solution logic faster, and they help developers troubleshoot extraction faster.

Major Unlock

Studio-first developers can now package useful complex Python logic, including coded agents and helper utilities, without needing to start as expert Python authors.

10-Min Demo

A fast path for a live walkthrough.

1
Open the presentation

Start with the full microsite and walk the audience through the lesson flow and the key message.

presentation/coding-agents-for-ixp/site/index.html
2
Run the offline coded-agent demo

Show the kind of Python review logic teams can now package much more easily.

python3 document_review_agent_demo/main.py --input document_review_agent_demo/input.sample.json
3
Walk the KSWIC payer-correspondence starter

Open one synthetic payer fax, then show the simulated IXP extraction and Maestro routing outputs.

demo_resources/kswic-payer-correspondence-demo/README.md
4
Run the evaluator smoke test

Show how extraction quality changes can be measured rather than discussed only anecdotally.

python3 document_ixp_sanity/main.py --run-smoke document_ixp_sanity/sample_ixp/sample1_ixp.json
5
Close on the platform message

Land the idea that coding agents expand both developer speed and developer capability.

Studio + IXP + coding-agent-assisted development + reusable Python logic
Repo Map

The repo is intentionally small and demo-safe.

presentation/coding-agents-for-ixp/

Slide deck, demo script, prompt pack, architecture handout, and the polished presentation site.

document_review_agent_demo/

Offline coded-agent-style demo showing field review against synthetic packet text.

document_ixp_sanity/

Generic evaluator package with sample data and smoke-run support.

demo_resources/

Synthetic packet text, extracted output, and ground truth used by the demos.

demo_resources/kswic-payer-correspondence-demo/

Synthetic payer-correspondence PDFs, IXP field contracts, Maestro flow assets, and dummy automation payloads.

GitHub Pages

This repo is ready for a simple branch-based Pages publish.

The root index.html is designed as a clean landing page for GitHub Pages. To publish it, set Pages to deploy from branch main and folder /(root). That will make this landing page the public front door to the demo without any Actions workflow.
